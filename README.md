# Expense Tracker JavaFX Application

A **desktop Expense Tracker** built with **JavaFX** and **MySQL**.  
Track expenses, manage recurring payments, set savings goals, and visualize spending with charts.

---

## Features

- Add, edit, and delete expenses
- Manage recurring expenses
- Set and track savings goals
- Update monthly income
- Search and filter expenses
- Visual charts: Pie, Line, and Bar
- Export expenses to CSV

---

## Technologies

- Java 17+
- JavaFX
- MySQL
- JDBC
- FXML & CSS
